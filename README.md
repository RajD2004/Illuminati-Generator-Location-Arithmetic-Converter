# C++ Mini Projects

This repository contains two short C++ programs:  
an ASCII “Illuminati” triangle generator and a location-string arithmetic converter.

## Files
- `main.cpp` – Illuminati Triangle Generator (ASCII art) :contentReference[oaicite:0]{index=0}  
- `locations.cpp` – Location Arithmetic Converter (string-to-decimal and back) :contentReference[oaicite:1]{index=1}  

## How to Compile & Run

### Illuminati Triangle Generator
Generates a triangle made of asterisks and inserts an eye `<O>` on the middle row.

```bash
g++ main.cpp -o triangle
./triangle
```
``` bash
g++ locations.cpp -o loc
./loc
```
